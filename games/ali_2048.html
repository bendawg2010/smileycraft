How To Play

Goal

Merge tiles to reach the legendary "ACTUALLY ATHLETIC" tile. (Spoiler: you probably won't.)

Controls

Desktop: Arrow keys or WASD
Mobile: Swipe in any direction

Rules

Tiles with the same milestone merge when pushed together. Each merge levels up Ali's athletic ability (somehow). New tiles appear after each move.

Undo

Ali gets ONE second chance per game. Use it wisely (he won't).
